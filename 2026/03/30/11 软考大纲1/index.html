客观题

1 计算机科学基础

1.1 数制及转换

2，8，10，16进制互转

1.2 数据的表示

最高位 = 符号位：0 = 正，1 = 负

机器数编码：

  • 原码，

  • 反码，正数同源码，负数符号位1，数值位按位取反。

  • 补码，正数同源码，负数符号位 1，反码 + 1

  • 移码，补码的符号位取反

数的表示：

  • 定点数，小数点位置固定
  • 浮点数，
  • 溢出。

编码：字符ASCll，汉字GB23112/GBK/UTF-8。

声音：采样，量化，编码。

图像：像素，分辨率，灰度/彩色，压缩。

校验码：奇偶校验，海明，CRC

1.3 算术和逻辑运算

二进制：加、减、乘、除、移位

逻辑运算：与、或、非、异或、同或

逻辑表达式化简：公式法、卡诺图

1.4 计算机系统结构

CPU：运算器、控制器、寄存器、Cache

存储器：寄存器→Cache→主存→外存（层次结构）

I/O 接口：数据缓冲、电平转换、控制逻辑、中断

虚拟存储：利用外存模拟内存，实现大容量寻址

多级存储：速度由快到慢、成本由高到低、容量由小到大

1.5 安全性、可靠性、性能评测

可靠性：MTBF 平均无故障时间、MTTR 平均修复时间

可用性：MTBF/(MTBF+MTTR)

容错：冗余、纠错、故障隔离

性能评测：基准测试、吞吐量、响应时间

2 嵌入式系统硬件

2.1 数字电路基础

组合电路：无记忆，输出仅由当前输入决定（与或非、译码器、编码器）

时序电路：有记忆，含触发器、计数器、寄存器

总线：数据 / 地址 / 控制总线，分时复用

电平转换：3.3V↔5V、TTL↔CMOS

可编程逻辑：CPLD、FPGA、GAL、PAL

2.2 嵌入式微处理器

2.2.1 体系结构

冯・诺依曼：指令、数据共用存储与总线

哈佛：指令、数据分开存储与总线（速度更快，嵌入式常用）

CISC：复杂指令集，x86

RISC：精简指令集，ARM、MIPS

流水线：取指、译码、执行、访存、写回

大小端：大端高字节存低地址；小端低字节存低地址

2.2.2 处理器类型

8 位：51、AVR、PIC

16 位：MSP430

32 位：ARM Cortex-M/A/R、MIPS

DSP：数字信号处理，乘累加快

多核：SMP 对称多核、AMP 非对称多核

2.2.3 异常与中断

同步异常：陷阱、故障、终止

异步异常：中断

可屏蔽 / 不可屏蔽中断

优先级、嵌套、中断向量、中断服务程序

2.3 存储体系

层次：寄存器→Cache→SRAM→DRAM→Flash→外存

Cache：高速缓冲，程序局部性原理

MMU：内存管理单元，虚拟地址↔物理地址

ROM

  • PROM：一次编程
  • EPROM：紫外线擦除
  • EEPROM：电可擦除，按字节

Flash

  • NOR：可片上执行，读快、容量小
  • NAND：块擦除，容量大、便宜，用于存储

RAM

  • SRAM：静态，速度快、功耗高、不需刷新
  • DRAM：动态，需刷新，密度大、常用做主存
  • DDR：双倍数据率
  • NVRAM：非易失 RAM

外存：SD、eMMC、UFS、磁盘、光盘

2.4 I/O接口

定时器 / 计数器：定时、计数、PWM 生成

GPIO：通用 IO，输入 / 输出 / 上拉 / 下拉

PWM：脉冲宽度调制，控电机、LED、蜂鸣器

A/D：模拟转数字，采样率、分辨率、精度

D/A：数字转模拟

人机接口：键盘、LED、LCD、触摸屏、音频

2.5 通信接口

低速：UART/USART、SPI、I²C、CAN

高速：USB、以太网、PCIe

无线：WiFi、蓝牙、ZigBee

其他：IrDA、1394、PCMCIA

2.6 电源

线性电源 LDO：噪声小、效率低

开关电源 DC-DC：效率高、噪声大

电源管理：降压、升压、稳压、休眠

2.7 电路与 PCB 设计

原理图：器件、连接、电气规则

PCB：布局、布线、阻抗、地平面、散热

多层板：电源层、地层、信号层分离

抗干扰：滤波、屏蔽、接地、隔离

3 嵌入式系统软件

3.1 嵌入式软件基础

分类：系统软件（OS）、支撑软件（驱动、中间件）、应用软件

无 OS 结构

  • 轮询：简单，实时性差
  • 中断：响应快
  • 前后台：后台主循环 + 前台中断

有 OS 结构：任务调度、资源管理

BSP 板级支持包：启动、驱动、硬件抽象

中间件：GUI、数据库、网络协议栈

3.2 嵌入式操作基础

结构

  • 单体：内核一体化，速度快
  • 分层：模块化
  • 微内核：服务用户态，稳定、开销大

任务管理

  • 任务：独立运行单元，有栈、控制块
  • 状态：就绪、运行、阻塞、挂起
  • 调度算法
    • 抢占式 / 非抢占式
    • FCFS、时间片轮转、优先级、短作业优先
    • 实时：RMS（速率单调）、EDF（最早截止）

同步互斥

  • 临界区：共享资源代码段
  • 信号量、互斥锁、事件、消息队列
  • 死锁：互斥、占有并等待、不可剥夺、循环等待

存储管理

  • Flat：无 MMU，直接物理地址
  • 分区：固定 / 可变
  • 页式：虚拟页、物理页、页表
  • 虚拟存储：请求调页、页面置换（FIFO、LRU）

设备管理

  • 轮询、中断、DMA
  • 缓冲、SPOOLING

文件系统：FAT、JFFS2、YAFFS2、EXT4

3.3 程序设计

语言：汇编、C、C++、Python

开发模式：宿主机 - 目标机

工具：编译器 GCC、链接器、调试器 GDB、仿真器 J-Link、逻辑分析仪

开发流程：设计→编码→编译→下载→调试→测试

驱动：字符设备、块设备、网络设备

3.4 开发与维护

生命周期：需求→设计→实现→测试→部署→维护

系统分析：数据流、功能分解、规格书

系统设计：总体、详细、软硬件划分

实施：编码、调试、集成

测试：单元、集成、系统、验收

维护：纠错、适应性、完善性、预防性

3.5 安全、标准、英语

安全：加密（对称 / 非对称）、认证、防火墙

标准：ISO、IEC、GB、行业标准

法规：知识产权、安全、隐私

专业英语：阅读手册、文档
